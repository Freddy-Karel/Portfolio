Application Météo

Consultez la météo en temps réel partout dans le monde

React Native Expo OpenWeatherMap Tailwind CSS

Présentation du Projet

Cette application mobile permet de consulter la météo en temps réel pour n'importe quelle ville dans le monde, avec une interface simple, moderne et intuitive.

Points clés

  • Données météo actualisées en temps réel
  • Interface utilisateur optimisée pour mobile
  • Sauvegarde des préférences utilisateur
  • Design moderne avec Tailwind CSS
  • Développement rapide avec Expo

Fonctionnalités Principales

Recherche de Ville

Recherchez n'importe quelle ville dans le monde pour obtenir les conditions météorologiques actuelles.

Villes Favorites

Sauvegardez vos villes préférées pour un accès rapide aux informations météo.

Prévisions Détaillées

Consultez les prévisions météo heure par heure avec température, humidité et vitesse du vent.

Mise à Jour Automatique

Les données se mettent à jour automatiquement lors de l'ouverture de l'application.

Interface Optimisée

Design responsive spécialement conçu pour les appareils mobiles avec une expérience utilisateur fluide.

Localisation

Détection automatique de votre position pour afficher la météo de votre emplacement actuel.

Interface Utilisateur

Écran Principal

Vue d'ensemble avec :

  • Recherche de ville
  • Accès aux favoris
  • Prévisions horaires
  • Températures actuelles

Détails Météo

Informations complètes :

  • Conditions météo actuelles
  • Température précise
  • Vitesse et direction du vent
  • Humidité et pression atmosphérique

Évolutions Possibles

Prévisions sur 7 Jours

Ajout d'un onglet pour afficher les prévisions météo à moyen terme avec des graphiques intuitifs.

Géolocalisation

Détection automatique de la position de l'utilisateur pour afficher la météo locale sans recherche.

Mode Sombre

Implémentation d'un thème sombre pour une meilleure expérience nocturne.

Notifications

Alertes météo personnalisables pour les événements importants (pluie, neige, etc.).
